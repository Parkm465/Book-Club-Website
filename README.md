# Book-Club-Website

## Story Summit Marathon

A static event website built as part of the CodePath Web Development course.

## Features
- Dark mode and reduced motion toggles
- Interactive RSVP form with validation
- Dynamic participant list and count
- Success modal with animation
- Responsive layout using Flexbox

## Tech Stack
- HTML5
- CSS3 (Flexbox, CSS variables)
- Vanilla JavaScript (DOM manipulation)
